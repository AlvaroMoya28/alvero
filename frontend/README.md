# Vero Frontend

Página romántica para celebrar el primer aniversario con Verónica.

## Instalación

```bash
cd frontend
npm install
```

## Ejecución

```bash
npm run serve
```

La aplicación se levantará en `http://localhost:8080` y también estará disponible en la IP local de tu equipo,
por ejemplo `http://192.168.x.x:8080`, para que Verónica pueda abrirla desde su dispositivo en la misma red.

## Sin base de datos

Esta web no usa ninguna base de datos. Todo el contenido romántico y los mensajes se muestran en el navegador sin conexión a un servidor de datos.

## Características incluidas

- Página romántica con estilo pastel y diseño moderno.
- Sección de historia, galería y jardín dedicada a Verónica.
- Carrusel de fotos automátizado.
- Música suave de fondo controlable desde la página.
- Panel interactivo de notas que guarda los mensajes en el navegador.
- Navegación interna para recorrer secciones de forma fluida.
